# StudyPlanManager

**StudyPlanManager** is a complete project for managing personal or educational study plans.  
It helps you organize your learning goals into **Plans**, **Weeks**, and **Daily Tasks**,  
with a simple and modern web interface.

---

## Note
> **This project is currently under active development.**  
> The current codebase is an **early-stage prototype** — I’m gradually expanding features, improving the architecture,  
> and refining both backend and frontend modules.
> Expect frequent updates and changes as the system evolves.  

---

## Tech Stack

---

###  Backend
- **ASP.NET Core 9 (C#)** — RESTful API.
- **Entity Framework Core (SQLite)** — Lightweight and efficient database.
- **JWT Authentication** — Secure user login and access control.

### Frontend
- **Vue 3 + TypeScript**
- **Vite** for fast development
- **Axios** for API calls
- **Bootstrap 5** for responsive UI

---

## Current Features
- User login and JWT authentication  
- CRUD operations for Plans, Weeks, and Tasks  
- Basic layout with sidebar navigation  
- API communication between Vue and .NET Core
